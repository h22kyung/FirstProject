# < REST API 기초 >
 
### 1. REST API 란?
-> 서버의 자원을 클라이언트에 구애받지 않고 사용할 수 있게 하는 설계 방식입니다.
-> HTTP 요청에 대한 응답으로 서버의 자원을 반환합니다.
-> 서버는 클라이언트의 요청에 대한 응답으로 화면이아닌 데이터를 전송합니다. (JSON 형식)

### 1-1. REST API 동작 살펴보기
https://jsonplaceholder.typicode.com/ 연습용 REST API 서버
